- **Monospace:** <e.g., JetBrains Mono, ui-monospace>

## Type scale

Modular ratio: <e.g., 1.250 — Major Third>. Base: 16px.

| Token | Size | Weight | Line height | Use |
|---|---|---|---|---|
| `heading-xl` | 48px | 700 | 56px | Page-hero title |
| `heading-lg` | 32px | 700 | 40px | Section title |
| `heading-md` | 24px | 600 | 32px | Card title / dialog title |
| `heading-sm` | 20px | 600 | 28px | Inline group title |
| `body-lg` | 18px | 400 | 28px | Lead paragraph |
| `body-md` | 16px | 400 | 24px | Default body |
| `body-sm` | 14px | 400 | 20px | Secondary text, captions |
| `label-md` | 14px | 600 | 20px | Form label, button text |
| `label-sm` | 12px | 600 | 16px | Tag, badge, micro-label |
| `code-md` | 14px | 400 | 20px | Inline code |

## Rules of use

- <e.g., Page never has more than one `heading-xl`>
- <e.g., Button text MUST use `label-md`; never `body-md`>
- <e.g., Form labels MUST use `label-md`; never `body-sm`>
